About Roles

Ticket Manager

Responsible for managing tickets, including sorting, addressing, and closing support tickets from players or community members.

Whitelist Manager

The Whitelist Manager controls access to the server by reviewing and granting permission to players, ensuring only trusted individuals can join.

Discord Staff

Responsible for managing the Discord server, overseeing community interactions, and assisting with server moderation and promotions.

Server Admin

Admins have full control over the server, making decisions, handling rules, and managing server configurations and settings.
